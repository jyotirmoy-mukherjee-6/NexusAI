# NexusAI

NexusAI is an enterprise-grade Agentic AI platform that intelligently routes natural language queries to the appropriate backend capability.

Depending on user intent, the system can:

• Search enterprise knowledge bases using Retrieval-Augmented Generation (RAG)
• Query structured business data using SQL
• Generate Machine Learning predictions
• Combine multiple tools into a single response

The project demonstrates enterprise AI architecture using modern Python frameworks.
